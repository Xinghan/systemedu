# SystemEdu - Feature Backlog

> 功能待办追踪。每轮开发回顾中用户确认保留的建议记录在此。
> 开始开发前先在 PRD 中补充详细设计。

## Admin & 知识树管理

- [x] 知识树可视化预览 — Admin 导入知识树后，提供预览端点返回树的图结构（节点+边），方便管理员可视化查看
- [x] 知识树导出功能 — 将现有项目的知识树导出为 JSON，方便在项目间复制和备份
- [x] 项目模板系统 — 管理员可将项目标记为"模板"，创建新项目时基于模板快速复制知识树结构
- [x] 知识树 AI 生成 — 输入项目信息后由 Qwen 自动生成知识树 JSON，支持粗/中/细三档颗粒度
- [ ] 生成后可视化节点编辑器 — 对 AI 生成的知识树提供拖拽式可视化编辑，支持调整顺序、修改属性、添加/删除节点，非技术人员也能方便操作
- [ ] 知识树模板库 — 将优质的 AI 生成树保存为模板，同类项目可一键复用，减少重复生成
- [ ] 多轮迭代优化 — 允许用户对已生成/已导入的知识树提出修改意见，AI 在现有基础上增量调整而非全部重新生成
