要建蚁群行为数据集，先得挑出 能贴标签又不咬人 的安全蚂蚁。认错种会被火蚁咬，太小的种贴不上标签。这关教你像科学家一样查特征。 To build the ant dataset you must first pick ants that are safe and taggable. Misreading means fire-ant stings; tiny ants can't hold a tag. Learn to read features like a scientist.

2. 腰结 2 个 → 看触角Waist node = 2 → antenna
