账户信息 /account/new
PurpleAir 用账户绑定你拥有的 sensor。账号一旦创建就一直用。
用户名 username 3+ 字符,字母/数字
 
邮箱 email 用于激活和找回
 
密码 password 8+ 字符,含字母+数字
 
登记 Sensor /sensors/add
PurpleAir 用 MAC 地址唯一识别每个 sensor。设备背面贴纸上有。
MAC 地址 XX:XX:XX:XX:XX:XX
 
Sensor 型号 官方家族系列
 
固件版本 firmware >= 7.00
 
设置位置 /sensors/location
把右侧地图上的 pin 拖到你 sensor 的安装位置。位置一旦审核通过就不能随便改。
海拔 altitude (m) 0—4000
 
安装高度 height (m) 建议 1.5—3 m
 
坐标 (lat, lon) >> 在右侧地图拖 pin
 
填写元数据 /sensors/meta
这些 metadata 决定 sensor 在公共地图上怎么显示。要让别人能信任你的数据,描述要诚实。
安装环境 影响算法校正
室外 Outdoor
挂阳台/屋顶/院子,直接接触空气
室内 Indoor
放卧室/教室内,数据不进公共地图
 
可见性 visibility 公开 = 上 PurpleAir 全球地图
公开 Public
所有人能看,记者/研究人员能引用
私有 Private
仅自己看,数据不进公共数据集
 
节点名 label 3+ 字符,地图上会显示
