向量组装师
你会把一组脑波数值看成空间里的一支箭头了。一个时刻 = 一个点 = 一支箭头；电极越多，箭头活在越高的维度里。

下一站 CSP 就靠旋转这些箭头找到最能分开"左手"和"右手"的方向 — 你刚搭好的，正是 CSP 要转的那捆箭头。
