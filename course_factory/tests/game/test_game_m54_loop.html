这一环转起来才能脑控走路
脑控系统是个闭环：脑波被读进来 → 解码器分类 → 映射成走路指令 → 送进 Minetest，角色一动又产生新脑波。任何一环断了，整条流就卡死，角色就僵在原地。
现在闭环卡住了。像调试工程师一样：点环节插探针，找断点，拖对应修复。
