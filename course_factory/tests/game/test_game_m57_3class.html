想停就停才好操作
你的解码器只会"左/右"——它永远从这两个里挑一个。所以你想放空发呆时，它照样硬判一个方向，角色就乱动。
这一关给管线加第三类"静息"：拖入静息采集块、把 CSP 改成 OVR、换 3 类分类器、在发指令前加 if 静息: hold 守卫。装好后，你放空时角色才会老老实实停住——这是脑控好操作的关键一步。
