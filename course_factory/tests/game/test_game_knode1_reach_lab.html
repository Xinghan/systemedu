通行判定规则

• 平地：总是可走

• 斜坡：坡度 ≤ 你的"爬坡能力"才可走

• 松软：陷入深度 ≤ 你的"抗陷能力"才可走

• 深坑：除非连续宽度 ≤ 跳跃高度，或架了桥

• 水：除非架了桥

• 悬崖：除非挂了绳

欢迎来到可行域实验室

你的任务是弄清楚"哪里能走哪里不能走"——调整小人的能力，或在地图上放置工具，让小人能从起点走到旗子。

1. 左侧三个滑杆是小人的能力：跳跃、爬坡、抗陷。
2. 工具按钮让你在地图上放桥/绳/板，鼠标点地图放置，再点一次"擦除"可移除。
3. 点"派小人出发"——小人会自动寻路。没路就失败。
4. 每关会绿色高亮所有可达格子。学会看：每种地貌为什么能/不能走。
