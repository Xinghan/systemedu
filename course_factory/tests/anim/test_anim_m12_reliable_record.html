01
阻抗合格 Impedance OK
< 50 kΩ
软件里点 "测阻抗", 电极变绿才算贴牢; 不绿就重新涂导电膏 / 拨开头发。 Run impedance test in software; green = good contact. If not, re-apply gel / part the hair.
GOOD 全绿 < 50kΩ · 信号干净稳 All green < 50k · clean steady
BAD 红色 > 200kΩ · 漂移 + 假信号 Red > 200k · drift + fake spikes
02
无 50Hz 工频 No 50Hz Hum
50/60 Hz
看频谱图 50Hz 处有没有一根尖峰; 有就远离插线板 / 充电器 / 开陷波滤波。 Check the spectrum at 50Hz for a spike; if present, move away from power strips / enable notch.
GOOD 50Hz 处平坦 · 无尖峰 Flat at 50Hz · no spike
BAD 50Hz 一根高峰 · 淹没脑波 Tall 50Hz spike · buries EEG
03
幅值在 μV 级 μV-Scale Amp
10-100 µV
真脑波只有几十微伏; 若波形飙到几千 µV 满屏, 那是接触不良或运动, 不是脑信号。 Real EEG is tens of µV; if it hits thousands µV full-screen, it's bad contact or motion, not brain.
GOOD 10-100µV 小幅起伏 10-100µV gentle wiggle
BAD 满屏 / 削顶 · 几千 µV Clipping · thousands µV
04
闭眼 alpha 出现 Eyes-Closed Alpha
8-12 Hz
闭眼放松 10 秒, 看枕区频谱 8-12Hz 是否鼓起一个包; 这是脑波最可靠的活体证据。 Close eyes 10s; watch occipital 8-12Hz bump grow. Best living proof of real EEG.
GOOD 闭眼 alpha 明显涨起 Alpha clearly rises closed
BAD 睁闭眼无差别 · 可疑 No open/closed diff · suspect
05
眨眼伪迹可辨 Blinks Identifiable
~200 µV
故意眨眼 3 次, 前额电极应出现 3 个大尖鼓包; 能认出伪迹, 说明系统在如实记录。 Blink 3 times on purpose; frontal channels show 3 big bumps. Seeing them = honest recording.
GOOD 眨眼 = 可预测大尖峰 Blink = predictable spikes
BAD 眨眼无反应 · 没接通 No response · not connected
