git 5 步发布 · hover 看完整命令
local to GitHub · ~3 min
1
git init
把当前文件夹变成 git 仓库
cd purpleair-airquality-node
git init -b main
2
git add
把要发布的文件入暂存区
git add raw/ processed/ scripts/ docs/ notebooks/ README.md
3
git commit
写一条"快照说明"封存版本
git commit -m "v1.0 · 30d dataset"
4
remote add
绑定 GitHub 上空仓库地址
git remote add origin git@github.com:<you>/purpleair-airquality-node.git
5
git push
推到云上, 公网立刻能 clone
git push -u origin main
