记住这个核心观点
力不是让物体动起来的原因，而是让物体变速或变向的原因。

看到物体启动、加速或转弯，就知道有力在作用。
Force is not what keeps an object moving.
Force is what changes how it moves — speed or direction.
∴ F = ma   ·   build the machine.
