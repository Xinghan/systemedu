CS-01 / ICA 盲源分离

把混在一起的信号拆成几路独立来源

电极录到的每一路，其实是脑波、眨眼、肌电几个来源混在一起。ICA 假设这些来源彼此独立，反推出一组权重把它们一路路拆开。 Each electrode records several sources mixed together. ICA assumes the sources are independent and recovers them.
  • 鸡尾酒会：几个声源同时响
  • 每个电极 = 各来源的混合
  • 独立假设 → 求解混权重
  • 拆出独立成分 IC
