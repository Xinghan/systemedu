MTH-06 / ICA 盲源分离

把混在一起的信号拆成几路独立来源

每个电极录到的，都是脑波、眨眼、心跳几个来源按不同比例混在一起。ICA 假设这些来源彼此独立，连续旋转出一组解混权重，把混合一路路拆开。 Each electrode is several sources mixed in different proportions. ICA rotates to find an unmixing that peels them apart.
  • 几个独立来源同时在响
  • 电极 = 各来源的线性混合 X=A·S
  • 连续旋转解混方向 W
  • 每路锁定为单一独立成分
