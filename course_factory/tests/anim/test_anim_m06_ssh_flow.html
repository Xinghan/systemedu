B · 10 个 Linux 命令清单

今天先把这 10 个用熟, 别的以后再说

10 commands · keep them in your head
pwd WHERE
我现在在哪个文件夹
pi@pi:~ $ pwd /home/pi
迷路时第一条命令: print working directory
ls LOOK
看这个文件夹里有什么
$ ls Desktop sensor.py log.txt
加 -l 看详情, 加 -a 看隐藏文件
cd ENTER
进入某个文件夹
$ cd Desktop pi@pi:~/Desktop $
cd .. 回上一层, cd ~ 回家
mkdir NEW
新建一个文件夹
$ mkdir purpleair
"make directory", 项目第一步建工作目录
cat READ
把文件内容打印出来
$ cat log.txt 2026-05-27 ok
小文件直接看; 大文件用 less / tail
nano EDIT
在终端里直接改文件
$ nano sensor.py [ Ctrl+O 存, Ctrl+X 退 ]
最友好的命令行编辑器, 别用 vim 暂时
rm DELETE
删除文件 (没有回收站!)
$ rm old.log
rm -rf 是核武器, 三思再敲
cp COPY
复制一份文件
$ cp sensor.py sensor.bak
改代码前先备份, 不怕改坏
mv MOVE
改名 或 移动到别处
$ mv tmp.csv data/aqi.csv
同时管"重命名"和"挪窝"两件事
sudo ADMIN
以管理员身份执行
$ sudo apt update [sudo] password:
装软件 / 改系统时要前缀 sudo
