A · MNE + BCI IV 是什么
想分析脑电, 先有 工具 再有 数据 · MNE 负责读和算, BCI IV 是公认的练手数据
这一关你只做两件事: 把 MNE 装进 Python, 再把一个真实实验室的数据集读进电脑, 第一次看到脑电数据的真身。
mne = 脑电分析的 numpy
numpy 是数组运算的标准库 · MNE 就是脑电领域那个标准库, 读文件 / 滤波 / 分段全靠它。
BCI IV = 脑机接口界的 ImageNet
ImageNet 是图像 AI 公认的基准数据 · BCI Competition IV 就是脑机接口公认的练手 + 比拼数据集。

D · 读进来后 · MNE Raw 对象的三件套

    

      

        
Raw = info + data + events

        what raw_gdf actually holds
      


      

        

          

            

              .info
              dict-like
            

            

              元数据 · 电极名字 / 数量 / 采样率 250 Hz
              metadata · channel names, count, sfreq
            

          

          

            

              .get_data()
              ndarray
            

            

              真正的电压矩阵 · 形状 (n_channels, n_samples)
              the voltage matrix · (channels, samples)
            

          

          

            

              .annotations / events
              list
            

            

              事件标记 · 哪一刻是哪一类运动想象
              event markers · when each MI class happened
            

          

        


        
>>> raw = mne.io.read_raw_gdf("A01T.gdf")
>>> raw
<RawGDF | A01T.gdf,
  25 x 672528 # 25 ch x samples
>
>>> raw.info["sfreq"]
250.0
>>> raw.get_data().shape
(25, 672528)  # ch, samples
>>> # 672528 / 250 / 60 = 44.8 min
