欢迎来到坐标寻宝

这个游戏让你体会"坐标"为什么比"左边那个"更精确。

说坐标： 系统高亮一个目标，你要正确写出它的 (行,列) 才能收集。
读坐标： 系统给一个坐标，你点击网格上对应的格子。
约定很重要： 你可以切换 (行,列) 或 (列,行) 约定——同一个点在两种约定下坐标不同。
坐标从 0 开始数： 左上角是 (0,0)。
目标： 用正确坐标收集 10+ 个位置，理解坐标的本质。
