# 动物老师 & 背景场景 — Image Prompt 集

用途：为 SystemEdu 各项目生成统一风格的卡通动物老师角色和场景背景。

设计 DNA：
- 水彩 + 平面插画混合
- 暖色 storybook 调色板（terracotta / amber / sand / dusty teal / muted purple）
- chibi 比例（大头 + 圆眼 + 小高光）
- Pixar × Studio Ghibli 风格
- 角色透明背景 1:1，背景 16:9 不含角色

建议用法：先生成蜥蜴老师（prompt 1）作为风格锚点，再用它做 character reference / image-to-image 输入后续 prompt，保证 12 个动物风格统一。

## 三、扩展模板

新增动物老师时套用结构：

```
[动物 + 性格形容词] chibi cartoon [animal] teacher character,
[体型/毛色/特征],
big round eyes with golden highlight [+ 眼镜/帽子等],
[职业服装 + 道具],
[姿势 + 表情],
chibi oversized head proportions.
Soft watercolor and flat illustration hybrid style,
warm storybook palette of terracotta, amber, sand, dusty teal and muted purple,
painterly shading with subtle paper texture, gentle outline,
soft ambient light from upper left,
child-friendly Pixar-meets-Studio-Ghibli storybook vibe.
Centered full body, isolated on transparent background, no text, no watermark.
--ar 1:1 --style raw
```

新增背景套用结构：把 `chibi cartoon ... teacher character` 整段换成 `wide cinematic painterly storybook scene of [场景描述]`，加 `no characters, no text`，比例改 `--ar 16:9`。
